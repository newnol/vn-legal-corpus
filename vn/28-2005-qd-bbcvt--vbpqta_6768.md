# Quyết định 28/2005/QĐ-BBCVT

DECISION
Promulgating branch standards
Issued together with this Decision are five branch standards:
- TCN 68-229: 2005: Land mobile radio equipment having an antenna connector intended for the transmission of data (and speech) Ố Technical requirements;
- TCN 68-230: 2005: Land mobile radio equipment having an antenna connector intended primarily for analogue speech Ố Technical requirements;
- TCN 68-231: 2005: Land mobile radio equipment using an integral antenna intended for the transmission of data (and speech) Ố Technical requirements;
- TCVN 68-232: 2005: Land mobile radio equipment using an integral antenna intended primarily for analogue speech Ố Technical requirements;
- TCVN 68-233: 2005: Cellular mobile CDMA 2000 1X base stations Ố Technical requirements.
This Decision takes effect 15 days after its publication in "CONG BAO."- (Summary)
