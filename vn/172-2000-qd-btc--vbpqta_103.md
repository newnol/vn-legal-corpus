# Decision 172/2000/QD-BTC

DECISION No. 172/2000/QD-BTC of November 1, 2000 promulgating the Regulation on numerical codes for State budget-using units
State budget-using units� numerical codes are a sequence of figures stipulated according to an uniform principle for provision to each State budget-using unit. The State budget-using units� numerical codes are used to identify State budget-using units, enclosed with the management criteria of each unit in service of State budget management.
Each State budget-using unit shall be provided with only one State budget-using unit numerical code for the whole process from its establishment to operation termination. State budget-using units� numerical codes already provided shall not be re-used for other State budget-using units.
State budget-using units, being grade I, II or III-estimate unit or sub-grade III estimate units belonging to the central budget and provincial/district/commune budget levels, shall have all to register and declare with the finance bodies in order to be provided with State budget-using units� numerical codes.- (Summary)
